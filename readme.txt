windows'98中进入MS-DOS打开文件方式为：>qbasic clndr/MBF。参见clndr.bat。
请重新编写.bat文件，主要修改文件夹名称和所在位置。编写bat文件可在桌面点击它调用程序。

调用：<汉字>,<RUN>,<START>。(可点击)
调整字体大小(TT=10*16?)，使文字纵向对齐，把全部内容纳入屏幕内。
程序运行中不能使用鼠标。
退出正在运行的程序：<e>,<e>,<任意键>。
退出qbasic：<FILE>,<EXIT>。(可点击)
返回Windows：点击右上角的<X>。

本程序百项功能,可简推日期,普及历法知识,也可做综合运算。
声明：任何重要推算都必须另外作考证，包括考古研究!

-1383:2050年资料段,余为近似段。资料包括后人构拟内容，并非都是实历。
前841共和起纪年、前104汉武帝太初起月日渐近实历。
注意：古代列国、古今邻国农历都有差异。
-9998:9999年约20000年供普及立法知识用，不是精确的天文学推算。

格式：[*]虚拟、近似，[:]起止，[n_n/n]年月日。
积数行[&]_60日周期数。元点：今日，指定日，生辰，伊历，古历，本机元(-8996年)。
古历包括黄帝历、夏历、殷历、周历、鲁历和颛顼历。古历部分属于知识系统，不是真历。

±V农月_近似段实月计数，资料段月名计数，V>1有闰示闰。±V农年：近似段为±12月。
n多次_切换年序操作春节方式或月日方式。遇<闰月优先>顺向±1月后找年名。
